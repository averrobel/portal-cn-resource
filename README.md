# portal-cn-resource

本仓库用于归档和发布多个独立的 HTML 页面资源。这些页面不针对任何特定域名或网站，旨在提供通用、轻量的前端页面集合，便于参考、复用或快速部署。

## 项目简介

`portal-cn-resource` 是一个静态页面资源库，收录了若干独立完成的 HTML 文件。每个页面均为自包含的完整实现，不依赖外部服务或特定后端。仓库本身不提供任何导航、聚合或跳转功能，仅作为页面文件的存储与版本管理场所。

## 目录说明

```
.
├── pages/        # 存放所有独立 HTML 页面
├── assets/       # 页面通用的静态资源（CSS、JS、图片等）
└── README.md     # 本文件
```

- **pages/**：每个子目录或文件对应一个独立页面，页面命名清晰，内容自解释。
- **assets/**：可被 pages 中页面引用的公共资源，避免重复文件。

## 页面归档说明

- 所有页面均为静态 HTML，兼容主流现代浏览器。
- 页面内不包含跟踪代码、第三方资源或外部依赖（除非在页面内显式声明）。
- 每个页面在提交时附带简要的提交信息，说明改动内容。
- 历史版本通过 Git 分支与标签管理，可回溯。
